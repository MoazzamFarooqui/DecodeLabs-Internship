# Project Overview

Rule-Based Chatbots are one of the fundamental concepts of Artificial Intelligence. Instead of learning from data, they respond to predefined user inputs using decision-making logic.

This project implements a simple chatbot using Python's if-else statements. The chatbot continuously accepts user input, sanitizes the input, processes it using predefined rules, and returns appropriate responses. It also handles invalid commands and allows users to exit the program gracefully.

---

# Features

- Rule-based chatbot using if-else logic
- Handles greeting commands
- Responds to predefined project-related questions
- Continuous interaction until the user exits
- Input sanitization using string normalization
- Friendly responses for unknown commands
- Graceful program termination
- Keyboard interruption handling

# Program Workflow

1. Accept user input.
2. Sanitize the input by removing extra spaces and converting it to lowercase.
3. Compare the input with predefined commands.
4. Generate the appropriate response using if-else logic.
5. Display the chatbot's response.
6. Continue running until the user enters an exit command or interrupts the program.

# Usage

1. Run the Python program.
2. Enter one of the supported commands.
3. Receive the chatbot's response.
4. Continue chatting or type **exit** to terminate the program.

---

# Supported Commands

| User Input | Description |
|------------|-------------|
| hello | Displays a greeting message |
| hi | Displays a greeting message |
| hey | Displays a greeting message |
| greetings | Displays a greeting message |
| initialize | Initializes the chatbot |
| what is project 1? | Explains Project 1 |
| what are the core skills required? | Displays the required skills |
| explain the ipo model | Explains the Input-Process-Output model |
| exit | Exits the chatbot |
| quit | Exits the chatbot |
| terminate | Exits the chatbot |
| bye | Exits the chatbot |
